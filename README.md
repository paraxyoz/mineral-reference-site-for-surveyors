# Справочник минералов для геодезистов

Этот сайт представляет собой удобный справочник минералов с приятным интерфейсом. Он включает поиск по названию и фильтры по твердости, цвету и блеску.

## Функции

- **Поиск по названию**: Введите часть названия минерала в поле поиска.
- **Фильтры**:
  - Твердость: 1-3, 4-6, 7-10
  - Цвет: белый, красный, зеленый, синий, желтый, черный, прозрачный
  - Блеск: металлический, стеклянный, перламутровый, алмазный, тусклый
- **Карточки минералов**: Каждая карточка содержит название, фото, твердость, цвет и блеск.
- **Описание**: При наведении курсора на карточку появляется подробное описание минерала.

## Запуск

Откройте файл `index.html` в веб-браузере.

## Структура файлов

- `index.html`: Основная HTML-страница
- `styles.css`: Стили для интерфейса
- `script.js`: JavaScript логика для поиска и фильтров
- `data.js`: Данные о минералах

## Добавление новых минералов

Чтобы добавить новый минерал, отредактируйте файл `data.js` и добавьте объект в массив `minerals` с полями: name, image, hardness, color, shine, description.

## Troubleshooting

- Если сайт не загружается, убедитесь, что все файлы находятся в одной папке.
- Для фото используйте URL изображений или локальные пути.
